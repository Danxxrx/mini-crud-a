# Mini CRUD – Encja „Produkt”

Projekt wykonany w ramach części **A**.  
Celem było przygotowanie pełnego CRUD dla jednej encji (produkt) z użyciem:
- Node.js + Express,
- SQLite,
- prostego frontendu HTML + JS (fetch),
- REST API z poprawnymi kodami HTTP.
